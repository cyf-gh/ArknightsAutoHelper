# Arknights Auto Helper
> 明日方舟辅助脚本，保护你的肝。



## 快速部署

### 0.python环境

需要安装**python**与**pip**，详情百度。

### 1.安装依赖

```powershell
# powershell
. ./install_dependency.ps1
```

### 2.设置模拟器

* 使用直接兼容的夜神模拟器
* 模拟器安装路径应为 D:\\Program Files\\Nox\\bin\\ 若不是请到 config/config.py进行修改

* 分辨率调整为1280x720

### 3.运行脚本

#### 关键阅读文件

```
config/config.py		软件配置路径，ocr等设置
examples/examples.py	脚本案例与解析
click_location.py		主要添加例如关卡所消耗的理智等配置
```

用py运行

~~~powershell
py foo.py
~~~
